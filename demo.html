<ps-icon> Component Demo

This page demonstrates the features of the framework-agnostic icon library.

1. Default Sizing (Scales with Font Size)

When no size attribute is used, the icon defaults to 1em, scaling with the parent's font-size.

This icon is 16px tall.
This icon is 24px tall.
This icon is 32px tall. 
<div style="font-size: 32px;">
  <ps-icon name="cloud"></ps-icon> This icon is 32px tall.
</div>

2. Fixed Sizes (via size attribute)

Use the size attribute to set a specific, fixed size.

xs
sm
md
lg
xl
xxl 
<ps-icon name="user" size="xs"></ps-icon>
<ps-icon name="user" size="sm"></ps-icon>
<ps-icon name="user" size="md"></ps-icon>
...and so on.

3. Color Control

The icon inherits its color from the parent's CSS color property.

Inherited Red
Inherited Blue
Direct Green 
<!-- Color is inherited from parent -->
<div style="color: #e74c3c;">
  <ps-icon name="home" size="lg"></ps-icon>
</div>

<!-- Color is set directly on the element -->
<ps-icon name="home" size="lg" style="color: #27ae60;"></ps-icon>

4. Error Handling (Invalid Icon Name)

Requesting an icon that does not exist fails gracefully without breaking the page.

Requesting <ps-icon name="invalid-icon-name">: (Icon is blank, check the developer console for a 404 error). 
<ps-icon name="invalid-icon-name"></ps-icon>
